
# 🚨 Disaster Preparedness and Response Education System

A web-based platform designed to improve **disaster awareness and preparedness** in **schools and colleges** through **interactive learning modules, virtual drills, quizzes, and alerts**.

---

# 📌 Project Overview

Natural disasters such as earthquakes, floods, fires, and cyclones can occur anytime. Educational institutions often lack structured training and preparedness systems. This project provides a **centralized digital platform** to educate students and staff about disaster safety and emergency response.

The system includes **role-based dashboards** for **Admin, Teacher, and Student**, making disaster preparedness structured and easy to manage.

---

# 🎯 Objectives

* Provide disaster preparedness awareness
* Conduct virtual drills and simulations
* Track student preparedness and performance
* Allow teachers to assign modules and quizzes
* Enable admin to manage institutions and users
* Send alerts and emergency notifications

---

### 👨‍💼 Admin
